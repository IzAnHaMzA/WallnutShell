# Walnut Shell Website

Static multi-page business website for Walnut Shell, built with HTML, CSS, and JavaScript.

## Pages
- `index.html` (Home)
- `products.html` (Products)
- `about.html` (About)
- `contact.html` (Contact)

## Assets
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/img/black-truffle.svg`
- `assets/img/white-truffle.svg`

## Run Locally
Open `index.html` in any browser, or serve the folder with a static server.
